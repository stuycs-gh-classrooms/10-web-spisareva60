The Koch Curve

Basic curve: 


      koch_curve(t, 4, 20)

This curve has several modifications:
1. The line color has been changed to pink
2. The lines have been made thicker
3. The length of each line has been randomized 


            koch_curve(t, 4, 20)
          


        def koch_curve(t, depth, size):
    if (depth == 1):
        t.pencolor("pink")
        t.pensize(3)
        t.fd(size)
    else:
        t.pencolor("pink")
        t.pensize(3)
        reduction = random.randrange(1, 3)
        koch_curve(t, depth-1, size)
        t.lt(60)
        koch_curve(t, depth-1, size * reduction)
        t.rt(120)
        koch_curve(t, depth-1, size * reduction)
        t.lt(60)
        koch_curve(t, depth-1, size * reduction)

The Sierpinski Triangle


Basic triangle: 


              sierpinski(t, 5, 200)

This triangle has several modifications:
1. The width of each line has been randomized
2. The base case is drawn in pink, and the rest of the triangle is drawn in purple 


              sierpinski(t, 5, 200)
            


         def sierpinski(t, depth, size):
    if depth == 1:
        t.width(random.randrange (1, 5))
        t.color("pink")
        t.lt(60)
        t.fd(size)
        t.rt(120)
        t.fd(size)
        t.rt(120)
        t.fd(size)
        t.rt(180)
        t.color("purple")
    else:
        sierpinski(t, depth-1, size/2)
        t.fd(size/2)
        sierpinski(t, depth-1, size/2)
        t.bk(size/2)
        t.lt(60)
        t.fd(size/2)
        t.rt(60)
        sierpinski(t, depth-1, size/2)
        t.rt(120)
        t.fd(size/2)
        t.lt(120)

Trees

Basic tree: 


                tree(t, 4, 50, 20)

This tree has several modifications:
1. Each branch gets smaller in length
2. Each branch gets smaller in width
3. The angles are randomized
3. The bottom branch is blue, the middle branches are purple, and the top branches are pink


                tree(t, 9, 70, 15)
              


            def tree(t, depth, size, angle):
    if depth == 0:
        t.fd(size)
        t.pencolor("pink")
        t.bk(size)
        t.pencolor("purple")
    else:
        t.pencolor("blue")
        t.pensize(depth)
        t.fd(size)
        t.rt(angle)
        t.pensize(depth)
        tree(t, depth-1, size * 0.8, angle + random.randrange(-20,20))
        t.lt(2 * angle)
        t.pensize(depth)
        tree(t, depth-1, size * 0.8, angle + random.randrange(-10,10))
        t.rt(angle)
        t.bk(size)
